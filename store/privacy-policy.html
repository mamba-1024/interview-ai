隐私政策

AI模拟面试（InterviewAI） · 最后更新日期：2026年6月1日 · 生效日期：2026年6月1日

概述

AI模拟面试（以下简称"本扩展"）是一款 Chrome 浏览器扩展程序，旨在帮助用户通过 AI 技术进行模拟面试练习。我们非常重视您的隐私保护，本隐私政策将详细说明我们如何收集、使用和保护您的信息。

核心承诺：本扩展不会收集、存储或传输任何个人身份信息。所有面试练习数据仅保存在您的本地浏览器中，不会上传至我们的服务器。

我们收集的信息

本扩展在运行过程中会涉及以下数据：

数据类型 存储位置 用途
API 配置信息（API 地址、API Key、模型名称） Chrome 本地存储（chrome.storage.sync） 连接您配置的 AI 大模型服务
岗位描述（JD）文本 Chrome 本地存储（chrome.storage.local） 分析和生成面试题
AI 生成的面试题和分析结果 Chrome 本地存储（chrome.storage.local） 面试练习和结果回顾
面试回答和评分记录 Chrome 本地存储（chrome.storage.local） 面试反馈和改进建议
使用统计（今日/累计面试次数） Chrome 本地存储（chrome.storage.local） 展示使用频率
重要说明：以上所有数据均存储在您的浏览器本地，我们不会将任何数据上传至自有服务器。卸载扩展后，本地数据将被自动清除。

第三方服务

本扩展需要与您自行配置的 AI 大模型服务进行通信，以完成岗位分析和面试题生成等功能。涉及的第三方服务包括：

  • 用户配置 阿里百炼（DashScope） — 阿里云的大模型 API 服务，用于 AI 文本生成
  • 用户配置 OpenAI API — 可选的 AI 服务接口
  • 用户配置 DeepSeek API — 可选的 AI 服务接口
  • 用户配置 其他 OpenAI 兼容接口 — 用户可自行配置任意兼容 API

当您使用面试分析功能时，以下内容会发送至您配置的第三方 AI 服务：

  • 招聘网站的岗位描述文本（用于生成面试题和分析）
  • 您的面试回答文本（用于 AI 评分和反馈）
  • 面试历史记录（用于生成面试总结报告）

请注意，这些数据的使用受相应第三方服务的隐私政策约束。我们建议您查阅相关服务的隐私条款：

  • 阿里云百炼：隐私政策
  • OpenAI：Privacy Policy
  • DeepSeek：隐私政策

权限使用说明

本扩展申请以下 Chrome 浏览器权限，每项权限均有明确的功能用途：

权限 用途说明
sidePanel 在浏览器侧边栏中显示面试助手界面，提供岗位分析和面试练习功能
storage 保存 API 配置、面试数据和使用记录到浏览器本地存储
activeTab 读取当前活动标签页的内容，以提取招聘网站的岗位描述信息
scripting 在招聘网站页面注入内容脚本，用于显示悬浮操作按钮和提取岗位信息

主机权限（host_permissions）仅用于以下受支持的招聘网站：

  • BOSS直聘（zhipin.com）— 自动识别岗位描述
  • 拉勾网（lagou.com）— 自动识别岗位描述
  • LinkedIn Jobs（linkedin.com/jobs）— 自动识别岗位描述
  • 智联招聘（zhaopin.com）— 自动识别岗位描述
  • 前程无忧（51job.com）— 自动识别岗位描述
  • 猎聘网（liepin.com）— 自动识别岗位描述
  • 牛客网（nowcoder.com）— 自动识别岗位描述

此外，主机权限还包括对您配置的 AI API 服务域名的访问，以便发送分析请求并接收 AI 响应。

数据安全

  • 所有数据均存储在浏览器本地的 Chrome Storage 中，不经过任何中间服务器
  • 与第三方 AI 服务的通信均使用 HTTPS 加密传输
  • 本扩展不包含任何广告、跟踪脚本或数据分析代码
  • 您的 API Key 仅保存在本地浏览器中，不会被发送至除 AI 服务外的任何第三方

您的权利

您对自己的数据拥有完全的控制权：

  • 查看数据：您可以在扩展的设置页面查看当前保存的所有数据
  • 删除数据：卸载扩展程序将自动清除所有本地存储的数据
  • 清除数据：您也可以通过 Chrome 浏览器的"清除浏览数据"功能手动删除扩展存储的数据
  • 停止使用：您可以随时禁用或卸载本扩展

儿童隐私

本扩展不面向 13 岁以下的儿童。我们不会有意收集儿童的个人信息。如果您是家长或监护人，发现您的孩子使用了本扩展，请与我们联系。

隐私政策更新

我们可能会不时更新本隐私政策。更新后的政策将在扩展的商店页面公布，重大变更将通过扩展内的通知告知用户。
