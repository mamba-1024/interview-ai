## 3. 详细描述

---

面试准备最大的痛苦是什么？不知道会被问什么。

翻遍面经、刷完八股文，到了面试现场才发现：问题跟你准备的完全不一样。每家公司、每个岗位的面试重点都不同，泛泛的通用题库根本不够用。

**AI模拟面试** 帮你解决这个核心问题 — 它根据你正在投递的具体岗位，由 AI 量身定制面试题，并进行真人面试般的模拟练习。

### 工作方式

打开 BOSS直聘、拉勾、LinkedIn 等招聘网站，点击页面上的 AI 悬浮按钮，扩展会自动提取当前岗位的 JD 信息，分析核心技能要求和面试重点，生成 5 道针对性面试题。

点击"开始模拟面试"，进入沉浸式**视频面试间**。AI 面试官朗读题目，实时听取并转写你的口头回答；也可手动输入文字作答。AI 从内容深度、逻辑清晰度、表达流畅度三个维度即时评分，并给出改进建议。表现好的话，AI 还会像真实面试官一样继续追问。

面试结束后，生成完整的面试报告，包含综合评分、各维度得分和逐题反馈。

### 核心功能

**智能 JD 解析** — 自动识别主流招聘网站的岗位描述，提取岗位名称、技能要求、经验年限等关键信息，无需手动复制粘贴。

**AI 定制出题** — 根据具体岗位的技能要求和面试重点，生成技术题、项目题、场景题、行为题等多种类型，覆盖真实面试的考察维度。

**沉浸式模拟面试** — 视频面试间界面，AI 面试官逐题朗读提问，支持语音作答与实时转写，也可手动文字输入，模拟真实面试节奏。

**语音交互** — 浏览器 TTS 朗读题目；支持三种语音识别引擎（Web Speech API / Deepgram / 科大讯飞），边说边显示转写文字。

**AI 智能追问** — 如果你的回答足够好，AI 会基于你的回答内容提出追问，就像真实面试官一样深挖细节。

**多维评分反馈** — 每道题从内容深度、逻辑清晰度、表达流畅度三个维度打分，并指出回答的亮点和可改进之处。

**面试报告导出** — 面试结束后生成综合报告，包含整体评分、各维度雷达得分和逐题分析，支持导出为文本文件。

### 支持的招聘网站

BOSS直聘、拉勾网、LinkedIn Jobs、智联招聘、前程无忧、猎聘网、牛客网。

也支持手动粘贴任意岗位的 JD 文本进行分析。

### AI 模型配置

本扩展需要你自行配置 AI 大模型 API，支持：

- 阿里百炼（DashScope）— 推荐，国内访问稳定
- DeepSeek API
- OpenAI 兼容接口
- 任意 OpenAI API 格式的自定义服务

首次使用时在扩展内填写 API 地址、API Key 和模型名称即可。

### 隐私与安全

所有数据（面试记录、API 配置）仅保存在浏览器本地存储中，不会上传至任何第三方服务器。与 AI 服务的通信使用 HTTPS 加密传输。扩展不包含广告、跟踪脚本或数据分析代码。

详细的隐私政策请查看：[隐私政策链接]

## 4. 权限说明（提交审核时填写）

| 权限 | 说明 |
|---|---|
| sidePanel | 在浏览器侧边栏中展示面试助手界面，用户可以在浏览招聘网站时随时打开面试工具 |
| storage | 将 API 配置和面试练习数据保存到浏览器本地存储，以便下次使用时加载历史记录 |

| scripting | 在支持的招聘网站页面中注入悬浮按钮脚本，提供快捷操作入口（提取JD、开始面试等） |

## 5. 商店图片素材规格

| 素材 | 尺寸 | 说明 |
|---|---|---|
| 扩展图标 | 128×128 px | 已有（icons/icon128.png） |
| 商店截图 | 1280×800 或 640×400 px | 至少 1 张，最多 5 张，PNG/JPEG |
| 小宣传图（Small Tile） | 440×280 px | 显示在商店列表页 |
| 大宣传图（Marquee） | 1400×560 px | 显示在商店详情页顶部（可选但建议提供） |

### 截图建议内容（5 张）

1. **悬浮按钮 + JD 提取** — 在 BOSS直聘页面上，悬浮按钮弹出菜单，展示"提取并分析 JD"功能
2. **侧边栏分析结果** — 侧边栏中显示的分析完成卡片，包含核心技能标签和面试重点
3. **面试题列表** — 新标签页中展示的 5 道面试题（最好用深色主题截图）
4. **模拟面试对话** — 面试进行中的对话界面，包含 AI 提问、用户回答和评分反馈卡片
5. **面试报告** — 面试总结页面，展示综合评分、各维度得分和逐题分析
